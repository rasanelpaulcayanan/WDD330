  2. Displaying/Hiding the Add a comment page

    Since the commentDiv is hard coded in HTML, I used the .style.display option in Javascript.
    It says “none” when all hikes is displayed and says “block” when the selected hike is displayed.

  3. Filtering the comments

    I got help from the renderOneHikeFull function, when that function is called a hike is passed though it.
    I used that function to call on the Addcomment Function which passes the name of the hike.
    This is then compared via for and if statements to the array that has all the comments.
    When the name on the array matches the name of the hike, the right comment is displayed.
