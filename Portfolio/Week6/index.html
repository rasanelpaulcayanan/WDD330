  2. Surprise local storage only takes in text

    This was tricky when trying to work on arrays. Since you have an object and array for the to do list, you can just pass
    the object and array to local storage. You can, but you wont be able to get anything out of it. So you have use JSON.stringify
    to convert your array to text, then pass it to local storage. You cant really do anything with a text converted array so you have
    to bring it back to an array by doing JSON.parse.

  3. taskList.completed was so much helpful for this program

    Another one of my challenges was the check box, it has to be changed when clicked. How do I do that for all the tasks that has already been completed?
    I created a task called changeImage that runs a loop through all the array items and if they are already completed, it changes the image. The array item
    is marked completed through the strike function. It underlines the text then marks the Completed variable true instead of the default false.

  4. All, Completed and Active

    I created loops then a if statement that if completed is true then it displays in completed. if completed is false it displays in the active tab.
