  3. The tricky page numbering

    It was hard to get the page numbers. The JSON file doesn’t really have that.
    What we did is we took the last character from the nextpage so if the next page
    URL shows as: https://swapi.dev/api/people/?page=2 we take out the last character of that string (2).
    So 2 is your next page, if you minus that by 1 then you get your current page.. (1).
